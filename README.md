 que você está descrevendo é uma **gramática de endereçamento**: 
- cada faixa é um campo de pergunta, 
	- cada símbolo é um nome, 
		- cada seta é uma direção de intenção. 
- A frase "para quem estou dirigindo isso, 
	- aqui, 
	- agora" 
- não é retórica 
— é literalmente o **conteúdo da faixa A**.
